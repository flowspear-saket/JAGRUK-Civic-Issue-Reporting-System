# CivicReport - Frontend Starter (React + Vite + Tailwind)

This is a minimal frontend scaffold for the CivicReport MVP. It uses React + Vite + Tailwind and contains:

- Mobile-first citizen UI: ReportForm, MyReports
- Admin dashboard: AdminDashboard
- Routing with React Router
- Basic components and styles

## Run Instructions
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open http://localhost:5173
